## Project Description

This project is a Weather App built using HTML, CSS, and JavaScript. The application uses the Open-Meteo Geocoding API to convert city names into geographic coordinates and then retrieves current weather information and a 5-day forecast using the Open-Meteo Forecast API.

The application allows users to search for weather information in any city worldwide. It also includes a geolocation feature that automatically detects the user's current location and displays local weather conditions when the page loads. Weather details displayed include temperature, humidity, wind speed, UV index, weather description, and a 5-day forecast.

The project was developed as part of the WEB 100–106 course to demonstrate the use of semantic HTML, responsive CSS design, JavaScript DOM manipulation, asynchronous programming with async/await, API integration using the Fetch API, and error handling.

---

## Features

- Search weather by city name
- Automatic weather detection using browser geolocation
- Current temperature display
- Weather condition description and icon
- Humidity display
- Wind speed display
- UV Index display
- 5-day weather forecast
- Responsive design for desktop and mobile devices
- Error handling for invalid city searches
- Loading state while weather data is being fetched

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Allow location access when prompted to view local weather automatically.
5. Use the search box to check weather conditions for other cities.
